# AGENDA — Agenda partagé

Un agenda partagé minimaliste inspiré des meilleures applications de calendrier (Apple Calendar, Fantastical).

## Fonctionnalités

- **3 vues** : Mois, Semaine, Jour
- **Personnes** : Chiara (rose bonbon) et Matéo (bleu), entièrement modifiables
- **Événements** : créer, modifier, glisser-déposer entre dates
- **Responsive** : desktop, tablette et mobile
- **Persistance** : tout est sauvegardé en `localStorage`

## Raccourcis clavier

| Touche | Action |
|--------|--------|
| `N` | Nouvel événement |
| `←` / `→` | Période précédente/suivante |
| `T` | Aller à aujourd'hui |
| `1` / `2` / `3` | Vue Mois / Semaine / Jour |

## Déploiement (GitHub Pages)

1. Créer un dépôt GitHub public
2. Pousser ce dossier
3. Dans *Settings → Pages*, choisir la branche `main` et le dossier racine `/`
4. L'agenda sera accessible à `https://<username>.github.io/<repo>/`
